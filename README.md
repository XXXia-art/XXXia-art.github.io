# Resume Template

Simple academic homepage template with separated content, styles, and interactive behavior.

## Structure

- `index.html`: page shell only
- `styles/main.css`: all visual styles
- `data/site-content.js`: profile and section content
- `scripts/main.js`: page rendering
- `scripts/interactive.js`: Pretext-based rotating mark layout
- `assets/`: local static files such as profile photos
